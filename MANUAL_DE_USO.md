# Manual de uso — Tata.Manager

Guía breve para empezar y trabajar día a día en **PC** y **celular**.

Podés abrir este manual **antes de iniciar sesión** (botón en el login) o desde el menú **Manual de usuario**.

---

## 1. Iniciar sesión

1. Abrí la app.  
2. Ingresá tu **usuario** y **contraseña**.  
3. Tocá **Entrar**.

### Si es tu primera vez
1. El **administrador** te crea la cuenta (usuario, email y contraseña temporal).  
2. Revisá tu **correo** (también spam).  
3. Entrá con la clave temporal.  
4. Si la app pide cambiarla, usá al menos **6 caracteres**.  
5. Completá **Mi perfil** (nombre, foto, email).

### Importante
- Usá un **email real**.  
- Para que PC y celular vean lo mismo, iniciá sesión en **cada** dispositivo.

---

## 2. Pasos recomendados de uso

### Al empezar el día
1. Iniciá sesión.  
2. Revisá la **pantalla de inicio** (stock, alertas, cuentas).  
3. Mirá el indicador de **sincronización** arriba (Sincronizado / Pendiente / Sin conexión).  
4. Si aparece **Sin stock**, tocá la tarjeta para ver esos productos.

### Sincronización entre PC y celular
- Con internet: los cambios se comparten solos entre dispositivos.  
- Sin internet: seguí trabajando; las ventas y cambios se guardan en el equipo.  
- Al volver la red: se envían solos (el indicador pasa a **Pendiente** y luego **Sincronizado**).  
- Tocá el indicador para ver el historial técnico; mantené pulsado para reintentar si hay error.

### Cargar o actualizar productos
1. **Productos** → alta o edición.  
2. Completá código, descripción, stock, costo y precios.  
3. Si importás por Excel: **Centro de importaciones** → **Importar productos**.  
   - Si cargás solo el **costo** y dejás precios vacíos, se calculan con el % de **Listas de precios**.

### Vender
1. **Venta rápida** o **Remitos**.  
2. Elegí cliente y productos.  
3. Confirmá: baja el stock y se genera el documento.  
4. El PDF queda en **Archivo PDF** para enviarlo después (útil desde el celular).

### Clientes y cobros
1. **Clientes** → alta o búsqueda.  
2. **Cuenta corriente** → ver deudas y registrar pagos.

### Compras y stock
1. **Compras** → registrar compra a proveedor (sube stock y actualiza costo).  
2. **Stock** → movimientos y alertas.

### Al cerrar el día
1. Revisá ventas / remitos del día.  
2. Enviá PDFs pendientes desde **Archivo PDF**.  
3. (Recomendado) hacé un **Respaldo** de vez en cuando.

## 3. Menú principal (resumen)

| Módulo | Para qué sirve |
|--------|----------------|
| Inicio | Resumen del negocio |
| Productos | Catálogo, precios, fotos |
| Venta rápida / Remitos | Vender y generar documentos |
| Clientes | Ficha e historial |
| Cuenta corriente | Deudas y pagos |
| Archivo PDF | Enviar remitos/facturas guardados |
| Stock / Compras | Inventario y compras a proveedor |
| Centro de importaciones | Excel de productos, clientes, costos |
| Reportes | Listados y exportaciones |
| Mi perfil | Foto, nombre, usuario, contraseña |
| Usuarios | Alta de personal (solo admin) |
| Configuración | Datos del negocio e impresión |
| Respaldo | Exportar / importar la base |
| Manual | Esta guía y el PDF |

---

## 4. Consejos rápidos

- No copies solo el `.exe` en Windows: llevá toda la carpeta **Instalador_Windows**.  
- En el celular, instalá el APK y permití orígenes desconocidos si lo pide.  
- Si no sincroniza entre equipos: verificá internet e inicio de sesión en ambos.  
- Si no llega el mail de alta: revisá spam y que el email esté bien escrito.
